Création d'une base de donnée

Conception de 0 à partir d'exigence du client.

Date
Janvier 2023
Équipe
Deux
Langage
SQL

Description

Le point de départ était une liste d'exigences fournies par le client. Nous avons dû filtrer les informations pertinentes afin de créer un schéma entité-association sur WinDesign, en respectant les différentes contraintes (unicité, contextuelle). Ensuite, nous avons créé la base de données et y avons ajouté des éléments, en utilisant le langage SQL. Pendant ce projet, la gestion de celui-ci était évaluée, ce qui nous a appris à utiliser un diagramme de Gantt pour planifier les différentes étapes.
